# Get-PodeSseDefaultScope

## SYNOPSIS

```
Get-PodeSseDefaultScope [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Retrieves the default SSE connection scope for new SSE connections.
