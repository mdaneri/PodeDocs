# Clear-PodeLogging

## SYNOPSIS
Clears the Pode logging.

## SYNTAX

```
Clear-PodeLogging
```

## DESCRIPTION
This function clears all the logs in Pode by calling the Clear method on the PodeLogger class.

## EXAMPLES

### EXAMPLE 1
