# Save-PodeSession

## SYNOPSIS
Saves the current Session's data.

## SYNTAX

```
Save-PodeSession [-Force] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Saves the current Session's data.

## EXAMPLES

### EXAMPLE 1
```
Save-PodeSession -Force
```
