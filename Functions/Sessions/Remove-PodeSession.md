# Remove-PodeSession

## SYNOPSIS
Remove the current Session, logging it out.

## SYNTAX

```
Remove-PodeSession [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Remove the current Session, logging it out.
This will remove the session from Storage, and Cookies.

## EXAMPLES

### EXAMPLE 1
```
Remove-PodeSession
```
