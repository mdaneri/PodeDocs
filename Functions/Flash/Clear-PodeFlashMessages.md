# Clear-PodeFlashMessages

## SYNOPSIS
Clears all flash messages.

## SYNTAX

```
Clear-PodeFlashMessages [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Clears all of the flash messages currently stored in the session.

## EXAMPLES

### EXAMPLE 1
```
Clear-PodeFlashMessages
```
